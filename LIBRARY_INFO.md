# Portfolio Book Library Information

## Library Used: **react-pageflip**

- **Package**: `react-pageflip`
- **Version**: `^2.0.3`
- **Description**: A React wrapper for creating interactive flipbook/page-flip effects
- **NPM**: https://www.npmjs.com/package/react-pageflip

## Usage in Project

The portfolio book uses `react-pageflip` to create a realistic book-flipping experience with:
- Page turning animations
- Two-page spread view (left and right pages)
- Smooth transitions
- Touch/mobile support

## Installation

Already installed in this project. If you need to reinstall:

```bash
npm install react-pageflip
```

## Documentation

For more information, visit: https://www.npmjs.com/package/react-pageflip
